# 🎧 GMPlayer (Web Music Player)

A simple browser-based music player built with HTML, CSS, and JavaScript.  
This project allows users to load and play local audio files directly in the browser.

---

## 🚀 Features (Current)

- Load local audio files from your device
- Play / Pause audio
- Stop and reset playback
- Basic UI status updates

---

## 🧠 Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Web Audio API (HTML Audio Element)
